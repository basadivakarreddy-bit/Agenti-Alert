# 🚀 Agenti Alerts

> **Empowering proactive professionals with critical opportunities and real-time updates.**

Agenti Alert is a sleek, modern opportunity discovery platform that aggregates and delivers the most relevant career, academic, and professional opportunities — from hackathons and internships to scholarships, remote jobs, and tech news — all in one place.

---

## ✨ Features

- 🔔 **Real-time Tracking** — Never miss a deadline with instant notifications
- 🎯 **Curated Content** — Quality opportunities vetted by our expert team
- 🌍 **Global Reach** — Access opportunities from all over the world in one place
- 🔐 **Secure Authentication** — Firebase Google Authentication for seamless login
- 📊 **Smart Dashboard** — Track and manage your saved opportunities effortlessly
- 🌗 **High Contrast Mode** — Toggle-able theme for better accessibility

## 🚀 Getting Started

### Prerequisites

- A modern browser (Chrome, Firefox, Edge)
- A Firebase project with Google Authentication enabled

### Running Locally

1. **Clone the repository**

   ```bash
   git clone https://github.com/basadivakarreddy-bit/Agenti-Alert.git
   cd Agenti-Alert
   ```

2. **Open in browser**
   Simply open `index.html` in your browser, or use a local development server:

   ```bash
   npx serve .
   ```

   Then navigate to `http://localhost:3000`

3. **Configure Firebase**
   - Go to [Firebase Console](https://console.firebase.google.com/)
   - Create a project and enable **Google Authentication**
   - Replace the Firebase config in `dashboard.js` and `auth.html` with your own credentials
